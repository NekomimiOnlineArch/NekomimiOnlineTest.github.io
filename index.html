Commands for the beta is enabled in all accounts:
  • @item - Spawns item id/name
  • @blevel - Increases Base level by number. Can use negatives as well.
  • @jlevel - Increases Job level by number. Can use negatives as well.
  • @spawn - Spawns monster.
  • @job - Changes to named job.
  • @zeny - Increases Zeny.
  • @killmonster - kills all monsters on field.
  • @go - Warp.
  • @warp - Warp to specific map.
  • @refine - Auto refines.

  • Swordsman

    Two-Handed Sword Mastery
    • Mastery ATK to +5 per skill level from +4
    • Increases HIT by +2 per skill level
    • Increases CRIT per level by +1 per skill level

    • Triple Base Regeneration (150 at level 10)
    • Increase Healing Item Recover bonus by 1% per 2 INT

    • SP requirement is now 2+1 per skill level (12 SP at level 10)
    Magnum Break
    • Range increased by 2 cell
    • SP reduced to 18 SP per cast
    Endure
    • Provides additional +5 Soft DEF per level
    • Increase endured hits by 7 + 1 per 2 levels
